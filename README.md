# 🎵 Music Store Sales Analysis (SQL Project)

## 📌 Overview

This project analyzes a music store database using SQL to uncover insights related to customer behavior, sales trends, and music preferences.

## 🎯 Objectives

* Identify top customers based on spending
* Analyze sales distribution across countries and cities
* Discover popular music genres and artists
* Evaluate overall revenue patterns

## 🛠️ Tech Stack

* SQL (PostgreSQL)

## 📂 Dataset

This project uses a publicly available music store dataset containing tables such as customers, invoices, tracks, albums, and artists.

## 📊 Analysis Performed

* Customer purchase analysis
* Country and city-wise sales insights
* Top genres and artists identification
* Revenue analysis using invoice data

## 📁 Project Files

* `music_store_database.sql` → Database structure and data
* `Music_Store_Query.sql` → SQL queries for analysis
* `Music_Store_Questions.pdf` → Business questions solved
* `README.md` → Project documentation

## 🚀 Key Insights

* Identified high-value customers contributing most revenue
* Found regions generating maximum sales
* Highlighted most popular music genres and artists

## 📌 Conclusion

This project demonstrates practical use of SQL for analyzing structured data and deriving business insights.
